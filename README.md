# Hi, I'm April 👋

Senior Product Manager based in Toronto. This profile is where I build in the open. Right now my focus is practical: figuring out what AI actually changes about the product manager's job, and proving it by shipping small things rather than just writing about them.

## What I'm exploring

I'm building prototypes and tools to pressure-test where AI is genuinely useful in a product workflow, not where it's hype. Two pieces are live:

**[NestList](https://github.com/aprilde/VibeCoding-NestList)** — an AI-powered registry builder I designed and coded to test a specific product bet: that the valuable signal in a registry is post-use utility ("did you actually use this?"), not purchase sentiment. It's a working Flask prototype with a recommendation engine, transparent reasoning as a UX pattern, and a ratings flywheel. The README walks through the product decisions behind each choice.

**[PM Stakeholder Agents](https://github.com/aprilde/pm-stakeholder-agents)** — Claude Code subagents that review a PRD, roadmap, or decision doc through the lens of a specific non-technical stakeholder, surfacing the objections you'd hit before you walk into the room. Two verticals are built and working (B2B SaaS and Marketplace).
